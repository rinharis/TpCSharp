1 Point2D

Création d'une classe

Créez un nouveau projet Java Exo02 dans Eclipse.

Si vous le faites sous Maven, pensez à ajuster votre pom.xml afin qu'il soit au moins en Java7 (cf pom de l'exo 01). N'oubliez pas de mettre à jour votre projet Maven dès que vous modifiez le fichier pom.xml.

Ecrire une nouvelle classe fr.dessin.Point2D dans le projet Exo02 qui devra respecter les contraintes suivantes :

  • 2 attributs x et y de type int
  • Une méthode pour modifier la valeur de x (void setX(int valX))
  • Une méthode pour modifier la valeur de y (void setY(int valY))
  • Une méthode pour récupérer la valeur de x (int getX())
  • Une méthode pour récupérer la valeur de y (int getY())
  • Une méthode pour afficher le point (void afficher()). Cette méthode affichera les valeurs des attributs x et y sur la sortie standard. Par exemple sous cette forme : 
    [4, 34]
  • Une méthode pour translater le point (void translater(int dX, int dY)). On est ici sur une translation vectorielle, la valeur de x devient x+dX et la valeur de y se transforme en y+dY.

Test

Ecrire une nouvelle classe fr.Run dans le projet Exo02 qui devra respecter les contraintes suivantes :

  • avoir une méthode public static void main(String[] args)
  • Dans la méthode main, créez deux variables locales qui seront des instances de fr.exo2.Point2D et appeler leurs méthodes.

Lancez votre classe (cf Exo01, Run As - Java Application)
