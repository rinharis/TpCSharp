1 Installation

Installation d'un serveur J2EE

Récupérez la version du Java adaptée à votre système (Windows, Unix, Mac ...)

Téléchargez la version adaptée à votre type de système 32 ou 64bits.

Prenez un JDK, car il va falloir compiler notre code Java afin de le lancer.

Téléchargement disponible ici http://www.oracle.com/technetwork/java/javase/downloads/index.html

Lancez l'installation

  • sous Windows, double cliquez sur le fichier, attention il faut avoir les autorisations d'administration
  • sous unix / linux, utilisez le RPM ou l'installateur de package dédié.

Modification des variables d'environnements

Afin de terminer l'installation il est conseillé de modifier vos variables d'environnements

  • La variable JAVA_HOME (en majuscule) pointera sur votre dossier d'installation du JDK. A titre d'exemple, sous windows 64bits : C:\Program Files\Java\jdk1.8.0_121
  • La variable PATH (en majuscule), doit contenir le chemin vers les binaires Java.

Sous Unix, il est conseillé de modifier le fichier .bashrc et d'y ajouter en fin de fichier :

export JAVA_HOME=/usr/java/jdk1.8.0._121/
export PATH=$JAVA_HOME/bin:$PATH

Pour valider que tout fonctionne, lancez un "Invité de commande" (un shell) et tapez les commandes : 

java -version
javac -version

Dans certain cas, il vous faudra redémarrer votre Windows avant que les variables d'environnements soient effectives.

Installation d'un éditeur de texte

Pour la suite nous allons avoir besoin d'un éditeur de texte simple.

Vous pouvez faire usage de notepad, vi, nano, ... ou installer un autre éditeur (Notepad++, Atom, ...)

Ne prenez jamais Word ou Wordpad, se sont des éditeurs qui traitent des fichiers propriétaires en binaires.

2 Première application

Création d'une classe Java

Créez un répertoire sur votre ordinateur : c:\temp\exo01, sous unix faites le dans votre répertoire personnel

Lancez un "Invité de commande" (un shell) et tapez les commandes :

  • Windows : 
    mkdir c:\temp
    puis 
    mkdir c:\temp\exo01
  • Unix : 
    cd
    puis 
    mkdir ./exo01

Dans le répertoire créez un fichier PremierExo.java (avec un 'P' et un 'E' majuscule)

Editez le fichier avec votre éditeur de texte

public class PremierExo {
  public static void main(String[] args) {
    System.out.println("Bonjour tout le monde");
  }
}

Sauvegardez votre fichier.

Compilation - En ligne de commande

Lancez un "Invité de commande" (un shell) et tapez les commandes :

  • Placez-vous dans le dossier que vous avez créé : 
    cd c:\temp\exo01
    ou 
    cd ./exo01
    sous Unix
  • Compilez : 
    javac PremierExo.java

Si tout se passe bien vous n'aurez pas de message lors de la dernière commande, sinon relisez le code.

Tapez la commande la commande suivante dans le dossier exo01 :

  • Windows : 
    dir
  • Unix : 
    ls

Vous constatez qu'un nouveau fichier, PremierExo.class est présent, c'est le résultat de la compilation de votre .java

Lancement - En ligne de commande

Pour tester notre programme nous allons le lancer en ligne de commande, lancez un "Invité de commande" (un shell) et tapez les commandes :

  • Si vous n'y êtes pas déjà, placez-vous dans le dossier exo01 (cd c:\temp\exo01 ou cd ./exo01)
  • Exécutez : 
    java PremierExo
    sans .class ou .java

Si tout se passe bien vous verrez sur la console

Bonjour tout le monde
