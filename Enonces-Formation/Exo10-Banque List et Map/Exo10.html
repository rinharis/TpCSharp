1 Fin du projet

Utilisation de l'interface List

Toujours dans le ProjetBank, nous allons modifier la mécanique interne de la classe fr.banque.Client

L'objectif est de faire usage de l'encapsulation afin d'avoir un impact null sur la classes fr.Run

Modifiez la déclaration de l'attribut qui contient la liste des comptes, c'est actuellement un tableau de compte. Transformez le en : java.util.List<Compte>.

Modifiez le code des méthodes qui en font usage, sans toucher aux prototypages. N'oubliez pas d'instancier votre liste quelque part (lors de sa déclaration ou dans le constructeur par exemple).

On indique ici le nom long de la classe afin que vous ne vous trompiez pas d'import, en effet il existe aussi une classe java.awt.List. 

...
import java.util.List;
import java.util.ArrayList;
...
public Client() {
  ...
  private List<Compte> listeCompte = new ArrayList<Compte>();
  ...
}
...

Pensez à regarder la documentation de l'objet java.util.List vous y trouverez le moyen de transformer une List<Compte> en Compte[] ce qui vous sera nécessaire dans les méthodes getComptes() et setComptes(Compte[] tab).

Validez vos changements dans votre main, qui ne doit pas changer (car nous n'avons pas touché aux prototypes des méthodes).
