Java - Exercice 12 - IO

Lecture / écriture de fichier en Java.

Nous allons réaliser un programme qui va lire les noms et les prénoms qui se trouvent dans deux fichiers textes et écrit un fichier d'utilisateur résultat.

1 Lecture-Ecriture

Lire les noms

Créez un nouveau projet Java standard (ou Maven).

Créez une nouvelle classe fr.io.NomPrenomReader, son rôle sera de lire les deux fichiers nom.txt et prenom.txt. Elle aura par exemple :

  • Les attributs de votre choix (aucun ou des constantes).
  • Les constructeurs qui vous semblent importants.
  • La méthode private static java.util.List<String> readFichier(String unNomDeFichier) throws IOException, qui s'occupe de lire tout un fichier et qui place chaque ligne dans la liste résultat.
  • La méthode public static java.util.List<String> readNom() throws IOException, qui fait appel à la méthode readFichier afin de récupérer tous les noms.
  • La méthode public static java.util.List<String> readPrenom() throws IOException, qui fait appel à la méthode readFichier afin de récupérer tous les prénoms.

Les fichiers nom.txt et prenom.txt sont dans le répertoire de l'énnoncé.

Important : chaque méthode est responsable de ses flux, n'oubliez pas de les fermer quoi qu'il arrive (utilisez un finally ou l'écriture sous forme de ressource)

Ecrire les utilisateurs

Créez une nouvelle classe fr.io.UtilisateurWriter dont le rôle sera d'écrire un fichier CSV qui sera formaté comme suit : numero;genre;nom;prenom. Elle aura par exemple :

  • Les attributs de votre choix (aucun ou des constantes).
  • Les constructeurs qui vous semblent importants.
  • La méthode public static void writeUtilisateur(String unNomDeFichier, java.util.List<String> desNoms, java.util.List<String> desPrenoms, int combien) throws IOException. Pour chaque ligne du fichier résultat, elle tirera au sort
    • un genre (Mr ou Mm)
    • un nom de la liste des noms
    • un prénom de la liste des prénoms
    Le paramètre combien représente le nombre de ligne à générer dans le fichier.
  • Rappel : pour mélanger une liste : java.util.Collections.shuffle(laListe). Pour tirer des valeurs (int ou boolean) au hasard : java.util.Random.

Exemple de fichier résultat :

0;Mm;Bourgeois;Rayan
1;Mm;Collet;Kevin
2;Mr;Delannoy;Matteo
3;Mm;Levy;Axel
4;Mm;Laroche;Noa
5;Mm;Buisson;Ethan

Tester

Créez une classe fr.Run avec une méthode main.

Faites usage de vos deux classes ainsi que de leurs méthodes et générez un fichier de 50 utilisateurs.
